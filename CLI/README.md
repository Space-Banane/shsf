# SHSF CLI

## Installation & Usage

### Run directly (no install):

```
npx shsf-cli --mode <mode> [options]
```

or with pnpm:

```
pnpm dlx shsf-cli --mode <mode> [options]
```

### Install globally:

```
npm install -g shsf-cli
# or
pnpm add -g shsf-cli
```

Then use anywhere:

```
shsf-cli --mode <mode> [options]
```

---

See `--help` for all options.
